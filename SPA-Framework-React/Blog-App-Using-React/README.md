# Blog App Using React

## Description
A React application that fetches blog posts using the Fetch API and displays them using a class component.

## Features
- Fetches posts from JSONPlaceholder API
- Displays title and body
- Uses componentDidMount()
- Uses componentDidCatch()
- Class-based React Components

## Technologies Used
- React JS
- JavaScript
- Fetch API

## Run

```bash
npm install
npm start
```
