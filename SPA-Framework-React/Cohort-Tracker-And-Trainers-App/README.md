# React Cohort Tracker and Trainers App

## Applications

### 1. Cohort Tracker
- Displays ongoing and completed cohorts
- Uses CSS Modules for styling

### 2. Trainers App
- Displays trainer details
- Uses React Router
- Home, Trainer List and Trainer Details pages

## Technologies Used
- React JS
- CSS Modules
- React Router DOM

## Run Application

npm install

npm start
