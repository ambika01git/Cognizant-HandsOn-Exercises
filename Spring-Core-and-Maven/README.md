# Module 5 – Spring Core and Maven

## Overview

This repository contains the solutions for **Module 5 – Spring Core and Maven** of the Cognizant Digital Nurture 4.0 Program.

The exercises cover the fundamentals of Spring Framework, including Dependency Injection (DI), Inversion of Control (IoC), Bean Configuration, Spring AOP, and Maven project management.

## Technologies Used

- Java
- Spring Core
- Spring AOP
- Maven
- Eclipse IDE



## Concepts Covered

- Spring Framework Basics
- Inversion of Control (IoC)
- Dependency Injection
- Constructor Injection
- Setter Injection
- Bean Configuration using XML
- Spring AOP
- Maven Project Structure





## How to Run

1. Import the project into Eclipse as a Maven Project.
2. Update Maven dependencies.
3. Run the application.
4. Verify the output.
