# Logging with SLF4J and Logback

## Exercise 1: Basic Logging

* Implemented basic logging using SLF4J Logger.
* Used error and warning log levels.

## Exercise 2: Parameterized Logging

* Implemented parameterized logging using SLF4J placeholders.
* Used dynamic values in log messages.

## Exercise 3: Different Appenders

* Configured Logback using `logback.xml`.
* Implemented ConsoleAppender and FileAppender.
* Generated application logs in console and file.

## Technologies Used

* Java
* Maven
* SLF4J
* Logback
* Eclipse IDE
