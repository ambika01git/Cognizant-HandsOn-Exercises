# Git HandsOn-2

## Objective

Learn how to ignore files and folders using the `.gitignore` file.

## Tasks Performed

- Created a `.gitignore` file.
- Created a `.log` file.
- Created a `log` folder.
- Configured Git to ignore `.log` files and the `log` folder.
- Verified the ignored files using `git status`.

## .gitignore Content

```text
*.log
log/
```

## Result

The `.log` files and the `log` folder were successfully ignored by Git.
